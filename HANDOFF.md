## コーディングルール

- **globals.cssの@themeブロック内のブランドカラーのみ変更する**
  - `--color-navy` → #1A2E44 に変更
  - `--color-navy-dark` → #0F1D2E に変更
  - `--color-accent` → #C84B2F に変更
  - `--color-accent-dark` → #A63D26 に変更
  - ※変数名は変えない（ユーティリティクラスが参照しているため）
- **それ以外のglobals.cssは編集しない**
- **Tailwind CSSを使用** - スタイルはユーティリティクラスで記述

## 最初にやること（セットアップ）

**※すでにdocフォルダとHANDOFF.mdが存在する場合（構成案プロンプトで作成済み）:**

```bash
# 1. docフォルダとHANDOFFを一時退避
cd /mnt/c/client_hp/sing
mv doc ../sing_doc_backup
mv HANDOFF.md ../sing_HANDOFF_backup.md

# 2. 一度親ディレクトリに移動してディレクトリを削除
cd /mnt/c/client_hp
rm -rf sing

# 3. テンプレートをクローン
gh repo clone tenchan000517/template-fullorder sing

# 4. docフォルダとHANDOFFを戻す
mv sing_doc_backup sing/doc
mv sing_HANDOFF_backup.md sing/HANDOFF.md

# 5. 依存関係インストール
cd sing
npm install
```
