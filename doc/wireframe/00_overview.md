### 1-1. マーケティング戦略（専門家1の視点）

#### コンバージョンファネル設計

**ターゲット1: 中小企業の経営者（製造業・建設業 / 40〜60代男性 / 東海エリア）**

```
認知 → 興味 → 比較検討 → 行動 → コンバージョン
  ↓      ↓         ↓        ↓          ↓
Google検索  実績・強み  他社比較   不安解消    フォーム送信
SNS広告    代表の想い  FAQ確認   無料相談訴求  電話問合せ
```

- **認知段階**:
  - 流入経路: Google検索「採用支援 愛知」「中小企業 採用 コンサル」「離職率 改善」、SNS（Instagram/YouTube経由）、経営者仲間からの紹介
  - 最初に見るページ: トップページ（65%）、サービスページ（25%）、採用情報ページ（10%）
  - この段階での訴求: 「採用して終わりにしない。活躍する人材を、定着する組織を。」
  - 離脱防止策: ファーストビュー3秒で「採用から定着・活躍まで一気通貫で支援する東海エリアの人財パートナー」であることを伝える

- **興味段階**:
  - 見るページ: サービスページ、代表挨拶ページ
  - 滞在時間目標: 各ページ2分以上
  - この段階での訴求: 「なぜ採用しても辞めるのか。その原因を、私たちは現場から知っています。」
  - 次のアクションへの誘導: 「まずは御社の課題を聞かせてください」でお問い合わせページへ

- **比較検討段階**:
  - 見るページ: サービス詳細、代表挨拶（信頼性確認）
  - 競合と比較される要素: 1.採用後の定着支援まであるか 2.自走できる仕組みを作れるか 3.東海エリアの実績があるか
  - この段階での訴求: 「採用会社・研修会社・コンサル会社が分かれていませんか？Singはすべてを一体で支援します。」
  - 不安解消コンテンツ: お客様の声（「現場をよく理解してくれる」「一緒に考えてくれるパートナー」）、具体的な支援フロー図

- **行動段階**:
  - 最後のハードル: 「費用が高そう」「コンサルって本当に成果が出るのか」「うちの業界に合うのか」
  - 背中を押す要素: 「まずは無料でご相談ください。御社の課題を一緒に整理するところから始めます。」、お客様の声の具体的エピソード、代表の人柄が伝わる写真
  - CTAの配置: 1.各ページ下部の固定CTA 2.サービス詳細の各セクション末尾 3.ページ最下部の大型CTAセクション

- **コンバージョン**:
  - 目標行動: お問い合わせフォーム送信 / 電話問い合わせ
  - フォーム到達率目標: 8%
  - フォーム完了率目標: 60%
  - 離脱防止策: 項目数7個以内、入力時間3分以内、「営業電話はしません」の明記

**サービスページ**:
- 社会的証明: お客様の声「現場をよく理解してくれる」「スピードが早い」 → 各サービスセクションに分散配置

- 親近性: 代表の人柄が伝わるエピソード、自然な写真 → 配置: メッセージ本文
- ストーリー: 人材業界での原体験から起業に至った想い → 配置: メッセージ本文

- 親近性: 社員の笑顔、チームの雰囲気 → 配置: 全体を通して

### 1-2. 採用ブランディング戦略（専門家2の視点）

#### 求職者インサイト分析

**若手求職者（20代・第二新卒〜若手社会人）の場合**:

優先順位:
1. 最重要: この会社で自分は成長できるか？（スキルが身につくか）
2. 重要: どんな人と働くのか？（社風、人間関係）
3. 重要: 仕事の面白さ・やりがいは？（単調ではないか）
4. やや重要: 給料・休日は？（生活できるか）
5. やや重要: 会社の将来性は？（長く働けるか）

対応策:
- 「成長できるか」→ 「入社半年で企業の採用プロジェクトを任された」という具体的エピソード。営業・企画・マーケティング・発信と幅広いスキルが身につくことを明記
- 「どんな人と」→ チームでの打ち合わせ風景、ランチの様子などリアルな日常写真。「体育会系」の良い面（勢いがある、仲間意識が強い、成長を後押しする文化）を具体的に伝える
- 「仕事の面白さ」→ 「経営者と直接仕事ができる」「自分の企画が企業の未来を変える」というスケール感を伝える
- 「給料・休日」→ 正直に記載。隠さないことで信頼を得る
- 「将来性」→ 2024年設立の若い会社だが、事業の社会的意義と成長性を伝える

**中途採用者（30代・経験者）の場合**:

優先順位:
1. 最重要: 自分の経験が活かせるか？（ポジション、裁量）
2. 重要: 会社のビジョンに共感できるか？（働く意味）
3. 重要: 年収・待遇は今より良いか？
4. やや重要: ワークライフバランスは取れるか？
5. やや重要: 会社の安定性は？

対応策:
- 「経験が活かせるか」→ 営業経験・人材業界経験・マーケ経験など、前職の経験別にどう活きるかを示す
- 「ビジョン共感」→ 代表メッセージと連動。「人をモノのように扱う業界への違和感」に共感する人は多い
- 「年収・待遇」→ レンジで正直に記載
- 「WLB」→ 「体育会系」と聞いて懸念する人への回答として、「勢いはあるが、働き方は合理的」という実態を伝える

#### 「入りたい」と思わせる感情設計

**社員インタビューの設計**:
- 目標感情: 「この人たちと一緒に働きたい」「こんな仕事がしたい」
- 文言の工夫:
  - NG例: 「やりがいのある仕事です」「成長できる環境です」
  - OK例: 「入社3ヶ月で、ある製造業の社長に直接プレゼンする機会をもらいました。緊張で声が震えたけど、『君の提案でうちの採用ページを作りたい』と言ってもらえた時は、本当に嬉しかった。」
- 写真の工夫:
  - NG例: スーツでかしこまったポートレート、無表情のオフィス写真
  - OK例: クライアント先で打ち合わせしている瞬間、チームでホワイトボードを囲んでいる場面、ランチタイムに笑い合っている自然な表情

**「体育会系」の正直な伝え方**:
- NG: 隠す、または「アットホームな職場」と嘘をつく
- OK: 「勢いがある」「全力で取り組む」「仲間を大事にする」というポジティブな文脈で伝える
- 具体例: 「確かにうちは勢いのある会社です。でもそれは、無理をさせるということではなく、一つひとつの仕事に全力で向き合う文化があるということ。だからこそ、成果が出た時の喜びも大きい。」

### 1-3. ブランド体験設計（専門家3の視点）

#### ブランドの「人格」定義

**もしこの会社が人だったら...**:
- 年齢: 30代前半（若くてエネルギッシュだが、考え方はしっかりしている）
- 性格: 情熱的、面倒見が良い、正直、行動が速い。根は真面目だが堅苦しくない
- 話し方: 熱量がある。丁寧だが、心に響く言葉を選ぶ。「一緒にやりましょう」が口癖
- 印象: 「この人と一緒に仕事がしたい」「この人なら本気で向き合ってくれる」

**この人格をデザインに落とし込む**:
- 色: ダークネイビー#1A2E44（誠実さ・深み）+ テラコッタレッド#C84B2F（情熱・温かみ・行動力）
- フォント: Noto Sans JP（モダンで読みやすい。ウェイトの使い分けで階層を作る）
- 写真: 現場の自然な表情。作り込まない。動きのある瞬間を切り取る
- 余白: たっぷり取る。詰め込まない。呼吸するように読める

**なぜこの色か**:
- ダークネイビー: 「紺色」ではなく「暗い藍色」。コンサルの信頼感を出しつつ、黒ほど冷たくない。経営者ターゲットに対して「任せられる」印象を与える
- テラコッタレッド: 赤の情熱と茶色の温かみの中間。「人を大事にする会社」の温もりと、「勢いがある会社」の行動力を同時に表現。紫やグラデーションといったAI的な選択を完全に排除した、人間味のある色
- この2色の組み合わせ: コントラストが強く、視認性が高い。日本の伝統的な色彩感覚にも馴染む

#### ビジュアル階層設計

**トップページのファーストビュー（視線の流れ）**:
```
1. メインビジュアル写真（画面全体）← 最初に視覚的インパクト
   ↓
2. メインキャッチコピー（大きな白文字、左寄せ）← 2番目に読む
   ↓
3. サブコピー（やや小さな白文字）← 3番目に意味を理解する
   ↓
4. CTAボタン（テラコッタレッド）← 最後に行動を促す
```

**各ページの視線の流れの原則**:
- Zパターン（左上→右上→左下→右下）を基本とするが、セクションごとに崩す
- 重要な情報は左上と中央に配置
- CTAは視線の終点（右下寄り or 中央下）に配置

#### 余白とリズムの設計

**余白の基準値**:
- セクション間の余白: 120px（PC）/ 80px（SP）※ただし内容の関連性に応じて80px〜160pxで変動
- 見出しと本文の余白: 32px（PC）/ 20px（SP）
- 段落間の余白: 20px（PC）/ 16px（SP）
- 要素グループ間の余白: 48px（PC）/ 32px（SP）
- ページ左右のパディング: 最大幅1200px、左右auto。内側パディング40px（PC）/ 20px（SP）

**余白の使い分け原則**:
- 関連する要素同士: 16px〜24px（近づけて関係性を示す）
- 同一セクション内の異なるブロック: 48px〜64px
- セクション間: 100px〜160px（内容の切り替わりを身体的に感じさせる）
- 特に強調したいセクションの前: 160px（一呼吸置かせる）

**リズムの設計**:
```
ヒーローセクション（100vh / 動的で引き込む）
  ↓ 余白120px
ミッション紹介（テキスト中心 / 静かに読ませる）
  ↓ 余白100px
事業概要（ビジュアルとテキストの交互 / 動きのあるレイアウト）
  ↓ 余白120px
強み（非対称レイアウト / 印象に残す）
  ↓ 余白100px
お客様の声（引用テキスト中心 / 信頼感）
  ↓ 余白80px
新着情報（コンパクトなリスト / 軽やかに）
  ↓ 余白120px
CTA（ダークネイビー背景 / 決断を促す）
```
