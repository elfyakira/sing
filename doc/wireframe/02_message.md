# 代表挨拶ページ 詳細設計

> 企業名: 株式会社Sing
> 作成日: 2026-03-06
> ファイル: 02_message.md

---

## ページ概要

- **目的**: 代表・清水駿之介の人柄と想いを伝え、「この人に任せたい」「この人の下で働きたい」という信頼感を醸成する
- **ターゲット**: 主に経営者（信頼できる人か確認したい）、次に求職者（どんな社長か知りたい）
- **CV導線上の役割**: 興味→比較検討（人物軸での信頼構築）
- **滞在時間目標**: 2分30秒
- **次のアクション**: サービスページ（経営者）/ 採用情報ページ（求職者）/ お問い合わせ（信頼獲得後の即行動）

### セクション2: 代表プロフィール＋メインメッセージ

**専門家1（マーケティング）の設計意図**:
代表の顔写真とプロフィールを最初に見せることで、「誰が語っているのか」を明確にする。人は情報よりも「誰が言っているか」で信頼を判断する。顔が見えることで、コンサルの怪しさへの不安を軽減する。

**専門家2（採用）の設計意図**:
求職者にとって「社長がどんな人か」は応募を左右する重要な情報。人材業界での原体験から起業した背景は、「この会社は人を大事にする」というメッセージとして強力に機能する。

**専門家3（デザイン）の設計意図**:
写真は大きすぎず小さすぎず、テキストとバランスの取れたサイズで配置。2カラムではなく、写真を上部に大きく配置し、その下にテキストを流す構成。写真の比率は3:4（縦長）で、代表の佇まいが伝わるポートレート。

**レイアウト**:
- 構成: 1カラム、中央揃え、最大幅800px
- 背景: #FFFFFF
- 余白: 上100px 下80px 左右auto
- モバイル時: 左右パディング20px

**コンテンツ**:

- 代表写真:
  - 内容: 清水駿之介の上半身〜膝上ポートレート。ジャケットは着ているがネクタイはなし。少し微笑んでいる自然な表情。背景はオフィスの一角か、窓際の自然光
  - サイズ: 幅360px × 高さ480px（PC）/ 幅100% × 高さauto（SP）、ただしSPは最大幅300px中央揃え
  - 位置: 中央揃え
  - 角丸: なし
  - 下余白: 40px

- 氏名:「清水 駿之介」
  - フォント: Noto Sans JP, Weight 700, 24px（PC）/20px（SP）, #222222
  - テキスト揃え: 中央
  - 下余白: 4px

- 肩書:「株式会社Sing 代表」
  - フォント: Noto Sans JP, Weight 400, 14px, #5A5A5A
  - テキスト揃え: 中央
  - 下余白: 48px

- メッセージ見出し:「「人が輝ける場所」を創る。それが私たちの原点です。」
  - フォント: Noto Sans JP, Weight 700, 26px（PC）/20px（SP）, #222222
  - テキスト揃え: 左
  - 行間: 1.5
  - 下余白: 32px

- メッセージ本文:

  「「人が活躍し、心から輝ける職場をつくりたい」。
  その強い想いが、株式会社Singの原点です。

  現在、日本全体で深刻な人手不足が叫ばれる一方で、
  数多くの採用サービスが乱立し、
  多くの企業様が「自社に本当に必要な解決策」を見出せずにいます。

  私は以前、人材業界に身を置いていた際に、
  離職率の高さや、人をモノのように扱う風潮に

  「採用できた」で終わる支援は、本当に企業のためになっているのか。
  入社した人が3ヶ月で辞めてしまう採用に、意味はあるのか。

  その問いが、Singの出発点です。

  だからこそSingでは、
  単なる「人手の確保」にとどまらず、
  入社後の「定着」と「育成」、
  そして企業の理念や真の魅力を可視化する
  「ブランディング制作」までを一気通貫で支援しています。

  「現在の人手不足」と「将来の人材不足」。
  その両方を根本から解決し、
  企業が自走し続けられる強い組織づくりを
  伴走支援することが私たちの使命です。

  企業と人の未来をつなぐ。
  私たちは、そのために走り続けます。」

  - フォント: Noto Sans JP, Weight 400, 16px（PC）/15px（SP）, #222222
  - テキスト揃え: 左

### セクション3: ビジョン・ミッション・バリュー

**専門家1（マーケティング）の設計意図**:
企業の方向性と価値観を明示する。経営者ターゲットは「この会社がどこに向かっているか」を見て、自社との相性を判断する。VALUEの頭文字が「PASSION」になるという仕掛けは、記憶に残るフックとして有効。

**専門家3（デザイン）の設計意図**:
MISSION/VISIONは大きなタイポグラフィで印象づけ、VALUEは7つの項目をコンパクトに並べる。全てを同じ重みで見せるのではなく、MISSIONを最も大きく、VISIONをその次、VALUEは控えめに。最後の「PASSION」の種明かしは、読み進めた人へのご褒美として小さく添える。

**レイアウト**:
- 構成: 1カラム、中央揃え
- 最大幅: 900px
- 背景: #F5F1EB
- 余白: 上100px 下100px 左右auto
- モバイル時: 左右パディング20px

**コンテンツ**:

**MISSION ブロック**:
- ラベル:「MISSION」
  - フォント: Inter, Weight 600, 13px, #C84B2F, 字間0.15em
  - 下余白: 16px

- テキスト:「未来を変える、未来へ繋ぐ、未来が輝く」
  - フォント: Noto Sans JP, Weight 700, 30px（PC）/22px（SP）, #222222
  - テキスト揃え: 中央
  - 行間: 1.5
  - 下余白: 64px

**VISION ブロック**:
- ラベル:「VISION」
  - フォント: Inter, Weight 600, 13px, #C84B2F, 字間0.15em
  - 下余白: 16px

- テキスト:「次世代へのバトンを繋いで、誰もが笑顔で活躍できる社会の実現」
  - フォント: Noto Sans JP, Weight 700, 22px（PC）/18px（SP）, #222222
  - テキスト揃え: 中央
  - 行間: 1.5
  - 下余白: 64px

**VALUE ブロック**:
- ラベル:「VALUE」
  - フォント: Inter, Weight 600, 13px, #C84B2F, 字間0.15em
  - テキスト揃え: 中央
  - 下余白: 32px

- 7つのバリュー（縦にリスト表示）:
  各項目の構成:
  - 頭文字: Inter, Weight 700, 20px, #C84B2F（例: "P"）
  - 英語名: Inter, Weight 600, 16px, #222222（例: "rofessional"、頭文字と結合して表示）
  - 日本語説明: Noto Sans JP, Weight 400, 14px, #5A5A5A
  - 項目間余白: 20px

  1. **P**rofessional「全てにおいて高い視座を持ち、素直で謙虚に考え行動しよう」
  2. **A**ggressive「主体性を持ち、仲間とパートナーの為に積極的な行動をしよう」
  3. **S**ing「頑張るを楽しむに変えて、人生を笑って楽しもう」
  4. **S**uccess「勝つべくして勝つ。必ず成功させる為に準備をしよう」
  5. **I**nnovation「昨日よりも今日。現状に満足せず、常に改革し続けよう」
  6. **O**bligation「自身の責務を理解し、自律して行動しよう」
  7. **N**exus「パートナーと仲間との繋がりを大切にし、感謝を伝えよう。そして次世代にバトンを繋ごう」

  - 下余白: 40px

- PASSION の種明かし:
  「すべての頭文字をつなげると ——」
  - フォント: Noto Sans JP, Weight 400, 14px, #5A5A5A
  - テキスト揃え: 中央
  - 下余白: 8px

  「PASSION」
  - フォント: Inter, Weight 700, 28px, #C84B2F
  - テキスト揃え: 中央
  - 下余白: 12px

  「誰よりも高い情熱を持って、笑顔を絶やさず明るく生きよう。」
  - フォント: Noto Sans JP, Weight 400, 15px, #222222
  - テキスト揃え: 中央

**アニメーション**:
- 「PASSION」の文字のみ: スクロール到達時にフェードイン、0.8秒、ease-out。他は全て静的表示。
